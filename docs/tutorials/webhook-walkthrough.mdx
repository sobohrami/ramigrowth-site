# Webhook walkthrough

1. Register `POST /v1/webhooks` with your HTTPS URL and events (`job.completed`, `job.failed`, `batch.completed`).
2. Store the signing secret; verify `X-Rama-Signature` on each request (raw body).
3. Return `2xx` quickly; process work asynchronously.

See the OpenAPI schemas `WebhookCreateRequest` and `WebhookDeliveryPayload` for exact fields.
