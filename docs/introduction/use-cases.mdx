# Use cases

## Content generation at scale

**Who uses it:** Publishing platforms, e-commerce, marketing teams.

Generate on-brand content programmatically — product descriptions, blog posts, email copy — with one job per request or batch pipelines.

## Smart document editing

**Who uses it:** SaaS editors, writing tools, CMS products.

Embed AI editing: “make this more concise”, “rewrite for a technical audience”, or structured diffs alongside edited text.

## Automated content pipelines

**Who uses it:** Newsletters, media, agencies.

Combine RAMA webhooks with your triggers (database events, schedules, queues) so content is generated without manual steps.

## AI features in your SaaS

**Who uses it:** Product teams shipping AI writing without owning model ops.

One API key, predictable per-operation pricing, and SDKs for Node.js and Python.
