# What is RAMA?

RAMA is an **AI automations API** built for developers who want to move fast. Instead of wiring raw model APIs yourself, RAMA provides purpose-built endpoints for common AI content workflows:

- **Content generation** — blog posts, product descriptions, social copy, and long-form articles from one API call
- **Smart editing** — rewrite, summarize, expand, or restyle existing content with structured instructions
- **Webhooks** — async jobs with signed delivery to your infrastructure

## Quick example

```bash
curl -X POST https://api.rama.ai/v1/content/generate \
  -H "Authorization: Bearer sk-rama-YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Write a 500-word blog post about AI automation trends in 2026",
    "format": "markdown",
    "tone": "professional"
  }'
```
