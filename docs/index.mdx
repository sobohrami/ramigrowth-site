## Canonical API details

The published OpenAPI spec lives at `openapi/openapi.yaml` in this repo. It is the source of truth for:

- **Auth:** Bearer tokens using RAMA API keys (`sk-rama-…`).
- **Rate limits:** Tier table (Free / Starter / Pro / Enterprise) in the spec description.
- **Webhooks:** Event strings `job.completed`, `job.failed`, `batch.completed`.
- **SDKs:** npm package `@rama/sdk` and PyPI package `rama-sdk`.
